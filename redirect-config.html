EGCHAT v2.0.0 - Configuración de Red

Direcciones IP del Servidor

Wi-Fi (Principal): 192.168.21.36:3002
Ethernet Virtual: 172.24.80.1:3002
WSL Virtual: 172.25.80.1:3002
Localhost: localhost:3002

Para acceder desde otros dispositivos:

  1. Asegúrate que el servidor EGCHAT esté corriendo en el puerto 3002
  2. Conecta los dispositivos a la misma red Wi-Fi
  3. Usa la dirección IP: 192.168.21.36:3002
  4. Si no funciona, prueba con las otras direcciones IP
  5. Para acceso externo, configura el firewall del servidor
Conectar a 192.168.21.36:3002 Conectar a Localhost

Si ningún enlace funciona, verifica que:

  • El servidor esté corriendo (npm run dev)
  • El firewall permita conexiones al puerto 3002
  • Los dispositivos estén en la misma red
